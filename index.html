Skill
HTML
HTML-стандартизированный язык разметки документов для просмотра веб-страниц в браузере. Веб-браузеры получают HTML документ от сервера по протоколам HTTP/HTTPS или открывают с локального диска, далее интерпретируют код в интерфейс, который будет отображаться на экране монитора.
CSS
CSS-формальный язык описания внешнего вида документа, написанного с использованием языка разметки. Также может применяться к любым XML-документам, например, к SVG или XUL.
FLEXBOX
Flexbox-представляет собой модель веб-макета CSS 3. Он находится на стадии рекомендации кандидата W3C. Гибкий макет позволяет автоматически размещать отзывчивые элементы в контейнере в зависимости от размера экрана.
ANIMATION
Animation-технические приёмы создания иллюзии движущихся изображений с помощью последовательности неподвижных изображений, сменяющих друг друга с большой частотой. Мультипликационное кино наряду с игровым является разновидностью художественного кино.
